# 乖宝自律管家
# Sweetie-Self-ManagementBuddy
一款帮助家长培养儿童自律习惯的 Android 应用，通过积分奖惩机制激励孩子完成任务。


## 功能特性

### 👨‍👩‍👧 家长端
- **任务管理**：创建单次/每日/周末/假期任务，设置不同评分等级的积分
- **评分确认**：审核孩子的自评记录，确认后积分生效
- **兑换审批**：审批孩子的兑换申请（钱/活动时长）
- **假期配置**：设置暑假、寒假时间
- **统计报表**：查看积分排行榜、消费统计等
- **多孩子管理**：支持管理多个孩子账户
- **多家长管理**：支持管理多个家长账户
- **密码管理**：修改自己的登录密码、重置孩子的登录密码

### 👧👦 孩子端
- **今日任务**：查看并完成任务自评（优良中差）
- **积分兑换**：申请兑换零花钱或活动时长
- **借分系统**：积分不足时可向家长申请借分
- **消费记录**：记录花钱/用时情况
- **成就系统**：解锁徽章、连续打卡奖励
- **个人档案**：查看个人信息和统计数据

- ## 默认账号

- 家长账号：`parent` / `123456`
- 孩子账号：`child_1` / `123456`
